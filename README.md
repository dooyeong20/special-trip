## 1. Getting Started

### 1-1. Environment

> -   JAVA 11
>-   MariaDB 10
> -   Gradle
>-   Ubuntu 20.04
> 

### 1-2. Installing / Cloning
`$ git clone https://github.com/dooyeong20/special-trip.git`

## 2. Build & Run App

### - Windows

    special-trip\triplus$ gradlew build (-x test)
    special-trip\triplus$ java -jar ./build/libs/{app}.war

### - MacOS / Linux
    special-trip\triplus$ ./gradlew build (-x test)
    special-trip\triplus$ java -jar ./build/libs/{app}.war
